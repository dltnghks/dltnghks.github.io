[11723번: 집합](https://www.acmicpc.net/problem/11723)


### 어떤 걸 사용하나요?


비트 마스크를 사용하면 된다.


![image](/assets/img/posts/2026-02-01-집합/caafb81686b5e8a0e87f18ca579f0a0d.png)


위 6가지 연산을 수행하는 프로그램을 작성해야 한다.


이를 위해 0000…0000으로 구성된 비트로 각각의 연산을 수행한다.


### 어떻게 구현하나요?


공집합 S를 가지고 연산을 수행한다.

- add x

    `S |= (1 << x)`

- remove x

    `S &= ~(1 << x)`

- check x

    `S & (1 << x)` 


    이 값이 0이면 해당 bit 는 없다.


    있을 때는 반환 값이 1이 아니다.

- toggle x

    `S ^= (1 << x)`

- all

    `S = (1 << 21) - 1`

- empty

    `S = 0`


```c++
#include <iostream>
#include <string>
#include <vector>

using namespace std;

unsigned int s;

void add(int num){
    s |= (1 << num);
}

void remove(int num){
    s &= ~(1 << num);
}

void check(int num){
    int n = s & (1 << num);
    if(n == 0)
        cout << "0\n";
    else
        cout << "1\n";
}

void toggle(int num){
    s ^= (1 << num);
}

void all(){
    s = (1 << 21) - 1;
}

void empty(){
    s = 0;
}

int main() {

    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);
    
    s = 0;

    int m;
    cin >> m;

    for(int i = 0; i < m; i++){
        string instruction;
        int num;
        cin >> instruction;

        if(instruction == "add"){
            cin >> num;
            add(num);
        }
        else if(instruction == "remove"){
            cin >> num;
            remove(num);
        }
        else if(instruction == "check"){
            cin >> num;
            check(num);
        }
        else if(instruction == "toggle"){
            cin >> num;
            toggle(num);
        }
        else if(instruction == "all"){
            all();
        }
        else if(instruction == "empty"){
            empty();
        }
        else{
            cout << "error\n";
        }
    }
    
    return 0;
}
```
